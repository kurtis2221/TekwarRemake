/===================================\
|TekWar Remake Alpha Preview version|
\===================================/

Missing features
================
- No Civilians, Police, Tek Goons, Level boss
- No Pickupable items
- Not all doors, sounds are added yet
- Some textures are still in 8 bit format

Current system requirements
===========================
- Windows XP SP2 or newer
- .Net Framework 2.0
- 512 MB RAM
- 2.0 Ghz CPU
- Nvidia 9500 GT

Controls
========
W,A,S,D - Movement
Left mouse - Equip weapon, Fire
1,2,3 - Change weapon
E - Use
Shift - Run
Space - Jump
C - Crouch
R - Mirror
H - Holster weapon
TAB - Automap
+,- - Automap zoom in/out
M - Music on/off

Graphics settings
=================
Bump maps - Enables simple depth textures
Parallax maps - Enables detailed depth textures
Post process - Enables: bloom, sun rays, ambient occlusion (dark edges)
Post quality - Post process quality
Volume - Changes the game's volume
Water fix - Disables reflective water
MIDI Music - When turned on it uses Sound Blaster OPL3 music
Grass - Enables grass
